ZT Shortcodes package includes: 

* ZT Shortcodes System 
* ZT Shortcodes Editors


# Features List

* Label
* Button
* Dropcap 
* Modal 
* Space
* Badge
* Message Box
* Font Icons
* Counter Circle
* Title / Heading 
* Tabs
* Block quote 
* Divider
* Testimonial
* Person
* Columns
* Pricing Table
* Gallery 
* Carousel 
* Social Icons
* Progress Bars
* Accordion / Toggle
* Video Play
* Google Map
* Boxes Content
* Counter Box
* Turning Text
* Flip Box
